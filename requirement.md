# 根据视频生成字幕APP

## 基本信息
- android系统运行，kotlin+gradle实现
- 输入：视频文件，来自UI选择
- 输出：字幕文件，保存为xxxx（视频文件同名）.srt，路径为应用的数据路径

## 核心功能
- 功能：根据输入的视频文件，先提取音频，再通过火山引擎接口调用，自动生成对应的字幕文件
- 提取出来的音频是mp3格式，单音轨
- 火山引擎接口代码样例：./java_client
- 火山引擎文档：https://www.volcengine.com/docs/6561/1354868?lang=zh , 内容为[大模型录音文件识别标准版API]
  - 接口中支持audio.url 与 audio.data 二选一方式上传音频，audio.data用的是base64编码
- 需要api key和access key 通过界面配置

## UI功能
- 视频选择按钮
- 开始转换按钮
- API配置按钮
- **日志显示区域**：实时显示处理过程的详细日志
  - 带时间戳的日志记录
  - 自动滚动到最新日志
  - 包含每个步骤的详细信息（视频选择、音频提取、API调用、任务轮询、SRT生成）
  - 错误信息包含详细堆栈跟踪
- **清除日志按钮**：清空日志显示区域
